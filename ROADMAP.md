# Roadmap

Future plans and development priorities for Baker Street Scenarios.

---

## Planned Improvements

### Enhanced Active Directory Attack Scenarios

**Status**: Planned

- Additional MITRE ATT&CK techniques (beyond the current 12)
- Multi-domain forest scenarios
- Azure AD hybrid scenarios
- Certificate-based authentication attacks (beyond ESC1)
- Advanced delegation attacks (Unconstrained, Constrained, Resource-Based)

### Automated Scenario Deployment

**Status**: Planned

- Ansible playbooks for infrastructure setup
- Terraform/Packer templates for VM provisioning
- Automated snapshot management
- Scenario reset automation
- Pre-flight validation scripts

### Advanced Attack Scenarios

**Status**: Planned

- Ransomware simulation scenarios
- Supply chain attack scenarios
- Living-off-the-land (LOLBAS) scenarios
- Cloud attack scenarios (AWS, Azure, GCP)
- Container/Kubernetes attack scenarios

### Scenario Replay and Reset

**Status**: Planned

- Automated scenario replay from snapshots
- Progress tracking and state management
- Multi-scenario orchestration
- Scenario versioning and rollback
- Performance benchmarking

### Integration Enhancements

**Status**: Planned

- Integration with Mythic C2 for advanced scenarios
- Traffic generator integration for realistic background noise
- SIEM/XDR product integration testing
- Automated detection validation
- Detection coverage reporting

---

## Long-Term Vision

- Self-service scenario portal
- Scenario marketplace and sharing
- AI-powered scenario generation
- Real-time scenario adaptation
- Advanced analytics and reporting
